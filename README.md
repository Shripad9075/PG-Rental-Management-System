# PG Rental Management System

## Project Description
The PG Rental Management System is a web-based application developed using Django. It helps users search, view, and book PG rooms online. PG owners can manage rooms and bookings, while the admin manages the complete system.

---

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

---

## Main Features
- User Registration and Login
- PG Room Management
- Room Booking System
- Admin Dashboard
- Complaint Management
- Review and Rating System
- Image Upload
- Booking History

---

## Modules

### Admin Module
- Manage Users
- Manage Rooms
- View Complaints
- Approve Reviews

### PG Owner Module
- Add/Edit/Delete Rooms
- Upload Room Images
- Manage Bookings

### User Module
- Search Rooms
- View Room Details
- Book Rooms
- Give Reviews and Ratings
